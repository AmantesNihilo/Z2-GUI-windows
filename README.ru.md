**Portable GUI для Windows: zapret2-пресеты, автотесты, запуск в один клик и tg-ws-proxy.**

Z2 GUI - portable-приложение для Windows, которое помогает подобрать рабочий пресет zapret2, протестировать его на нужных целях, сохранить удачные варианты в избранное и запускать выбранную конфигурацию без ручной работы с bat-файлами.

Приложение также умеет управлять локальным `tg-ws-proxy`: поднимать Telegram MTProto WebSocket proxy, копировать proxy-link и открывать его сразу в Telegram Desktop.

## Что Это

| Z2 GUI это | Z2 GUI не является |
| --- | --- |
| GUI-оболочка для zapret2, пресетов и runtime-инструментов. | VPN или туннелем всего трафика. |
| Локальный тестер и лаунчер пресетов. | Гарантией работы любого сервиса в любой сети. |
| Удобный способ управлять `winws2` и `tg-ws-proxy`. | Заменой понимания того, что именно запускается на ПК. |

> [!IMPORTANT]
> Z2 GUI не является VPN. Приложение управляет локальными инструментами обхода DPI и локальным Telegram proxy. Используйте его только там, где это не нарушает правила вашей сети и местное законодательство.

> [!WARNING]
> Для запуска zapret нужны права администратора: `winws2.exe` и WinDivert работают с сетевыми пакетами. Некоторые антивирусы могут реагировать на WinDivert, сетевые инструменты или PyInstaller-сборки. Скачивайте приложение только из доверенного источника.

> [!CAUTION]
> Не запускайте неизвестные пересобранные версии без доверия к автору сборки. Portable-папка содержит исполняемые файлы, драйверные компоненты, пресеты, логи и локальные настройки.

## Возможности

| Раздел | Что делает Z2 GUI |
| --- | --- |
| Presets | Таблица пресетов, поиск, фильтры, избранное, свои пресеты, скрытие и удаление. |
| Testing | Автотест выбранных пресетов по Discord, YouTube и другим целям из `Test Settings`. |
| Results | Сохранение результатов последнего теста и сортировка по `Score`. |
| Launch | Home-экран с избранными пресетами и общей логикой Start / Stop. |
| tg-ws | Локальный Telegram MTProto WebSocket proxy: порт, secret, копирование ссылки, Open in Telegram, Restart. |
| Activity | Две отдельные консоли: zapret и tg-ws. |
| Safety | Проверка конфликтов с VPN/proxy-клиентами перед тестом и запуском. |
| UI | Fluent-like интерфейс, темы Windows 11, Dark, AMOLED и Light. |
| Packaging | Portable-сборка, Python на целевом ПК не нужен. |

## Требования

- Windows 10/11 рекомендуется.
- Права администратора для zapret / `winws2`.
- Telegram Desktop, если нужен `tg-ws-proxy`.
- Доступ в интернет для реальных проверок целей.

> [!NOTE]
> Сам `tg-ws-proxy` не требует прав администратора. Но если запускать его вместе с zapret через Home-кнопку, приложение заранее проверяет права, чтобы не получить частичный запуск.

## Быстрый Старт

1. Скачайте или соберите portable-пакет.
2. Откройте распакованную папку.
3. Запустите exe. Windows должна автоматически показать UAC-запрос:

```text
Zapret2GUI.exe
```

4. Откройте `Presets` и добавьте один или несколько пресетов в Favorites через звездочку.
5. По желанию запустите `Test Selected`, чтобы найти лучший пресет по `Score`.
6. Вернитесь на `Home`, выберите избранный пресет и нажмите `Start`.

> [!TIP]
> Home специально показывает только Favorites. Так главный экран остается чистым, а случайный запуск из огромного списка пресетов становится менее вероятным.

## Зачем Нужен Администратор

zapret2 использует `winws2.exe` и WinDivert. Эти инструменты смотрят и изменяют сетевые пакеты, поэтому Windows требует повышенные права.

Без прав администратора можно открыть Z2 GUI, смотреть пресеты, менять настройки и настраивать tg-ws. Запуск или остановка zapret может не сработать без elevation.

## Тестирование Пресетов

1. Откройте `Presets`.
2. Выберите строки галочками в колонке `Test`.
3. Откройте `Test Settings` и оставьте только нужные цели, например только Discord или только YouTube.
4. Нажмите `Test Selected`.
5. Следите за процессом в `Activity -> zapret`.
6. Отсортируйте по `Score` и добавьте лучшие пресеты в Favorites.

Колонки результатов:

| Колонка | Значение |
| --- | --- |
| OK | Успешные проверки целей. |
| FAIL | Проваленные проверки. |
| UNSUP | Неподдерживаемые проверки. |
| Ping | Успешные ping-проверки / всего ping-проверок. |
| Score | Общая оценка пресета. Чем выше, тем обычно лучше. |

Последние результаты хранятся локально:

```text
utils/gui-results.json
```

## Избранное

Пресет становится избранным после нажатия на звездочку в колонке `Fav`.

Избранное хранится локально:

```text
utils/gui-settings.json
```

В dropdown на Home отображаются только Favorites.

## tg-ws-proxy

Z2 GUI использует source-based интеграцию с `tg-ws-proxy`. Он запускается как helper-процесс Z2 GUI, а не как отдельный скачанный exe.

### Как Включить

1. Откройте `TG`.
2. Проверьте порт. Endpoint по умолчанию:

```text
127.0.0.1:1443
```

3. Оставьте сгенерированный `Secret` или задайте свой 32-символьный hex secret.
4. Нажмите `Restart tg-ws`.
5. Нажмите `Open in Telegram` или `Copy proxy link`.

### Интеграция С Home

На Home есть карточка `tg-ws`. Переключатель `Start with main button` решает, будет ли tg-ws запускаться общей кнопкой `Start`.

Можно запускать только zapret, только tg-ws или оба вместе.

### Логи

Файл лога tg-ws:

```text
utils/tg-ws-proxy.log
```

Его можно открыть со страницы `TG` или посмотреть tail в `Activity -> tg-ws`.

## Логи Activity

Страница `Activity` содержит две консоли:

- `zapret` - события приложения, тесты пресетов, запуск/остановка winws2, проверка конфликтов;
- `tg-ws` - события tg-ws и последние строки из `utils/tg-ws-proxy.log`.

`Clear Log` очищает только текущую выбранную консоль.

## Проблемы И Решения

| Проблема | Что делать |
| --- | --- |
| zapret не запускается | Запустите Z2 GUI от имени администратора. |
| Пресет сразу завершается | Попробуйте другой пресет или проверьте файл пресета. |
| Тесты идут слишком долго | Откройте `Test Settings` и оставьте только нужные цели. |
| Discord или YouTube все равно не работает | Остановите текущий пресет, протестируйте другие, выберите более высокий `Score`. |
| Найден VPN/proxy conflict | Закройте указанный клиент из диалога или вручную. VPN/proxy-клиенты часто перехватывают трафик и ломают работу zapret. |
| tg-ws не стартует | Проверьте, не занят ли порт `1443`, или смените порт на странице `TG`. |
| Telegram не открывает proxy link | Используйте `Copy proxy link`, отправьте ссылку себе в Telegram и нажмите вручную. |
| Антивирус ругается | Проверьте исходники, при необходимости соберите сами, и помните, что WinDivert/network tools часто ловят heuristic-флаги. |

## Сборка Из Исходников

Установить зависимости:

```powershell
python -m pip install -r requirements.txt
```

Запустить из исходников:

```powershell
python .\gui\app.py
```

Собрать portable-пакет:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\build-exe.ps1
```

Результат:

```text
dist/Zapret2GUI/Zapret2GUI.exe
```

Держите portable-папку целиком. Если перенести только `Zapret2GUI.exe`, могут сломаться пресеты, WinDivert, иконки, логи, лицензии или tg-ws integration.

## Структура Проекта

```text
gui/                 GUI и логика приложения
gui/zapret_core/     presets, testing, winws2, tg-ws managers
presets/             встроенные пресеты
utils/               настройки, результаты, логи, runtime-файлы
exe/                 winws2 и WinDivert файлы
vendor/              vendored исходники сторонних компонентов
lists/               host/ip списки
lua/                 zapret Lua scripts
```

## Безопасность

Z2 GUI работает с сетевыми инструментами, поэтому относитесь к сборкам внимательно:

- используйте официальные или собственноручно собранные версии;
- проверяйте, что portable-папку не менял неизвестный третий человек;
- ожидайте heuristic-срабатывания антивирусов на WinDivert, `winws2`, packet tools или PyInstaller;
- закрывайте VPN/proxy-клиенты, если приложение сообщает о конфликте;
- не публикуйте свой `tg-ws` secret;
- проверяйте custom presets перед добавлением.
